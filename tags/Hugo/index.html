如何部署Hugo至GitHub Pages

2022年4月12日
先决条件 你已在本地部署 Hugo 你已拥有 GitHub 账号并下载了 Github Desktop 生成最终网页 hugo --theme=xxxxx --baseUrl="http://xxxxx.github.io/" BaseUrl后填写你的网址，也就是GitHub（xxxxx.GitHub.io）或你要绑定的域名 theme后填你使用的主题文件夹名 千万不要忘记在网址最后加上/，否则你会被debug折磨致死 配置 Github Desktop 双击 Github Desk……
